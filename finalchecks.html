Final checks

Some final checks you can perform to help ensure everything works as intended are:

  1. Ensure all PCs and users are displayed in Windows Server: Tools>Active Directory Users and Computers. From here you can find an option that would show you all the computers ever connected to your network
  2. Ensure you can access your Windows PC from Ubuntu
    1. Get the IP address of your windows PC using ipconfig
    2. On Ubuntu, place the address in your browser. If it doesn't work, check WAMP services, .conf files (to allow remote access), Windows firewall ports (you can disable it entirely for testing purposes), and the URL
    3. Note: after connecting both PCs to AD, you will likely need your Windows Server running for them to network properly
  3. Ensure both your Ubuntu and Windows 10 PCs can access the internet (showing your DNS & DHCP are all configured properly)
  4. Ensure your Joomla installation is customized and everything is backed up and restored on XAMPP
