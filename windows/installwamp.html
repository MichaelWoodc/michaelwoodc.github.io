Installing Wamp server

  1. Download all dependencies and install
    • All vcredist packages
  2. Check to make sure all dependencies are installed
  3. Download and install Wamp Server
Finishing up, you should
  1. Enable WAMP system services (So they autostart on windows logon)
    • Search for services in the task bar
    • Click services
    • Find all 3 services starting with "wamp", double click, and set startup type to "Automatic"
  2. Create a .bat file to autorun the Wamp manager on startup (Only if desired to autostart the GUI manager)
  3. Restart system and test by navigating to localhost. Everything should work as intended, although it may take some time for the services to start. You can run Wamp to check the status of services.
