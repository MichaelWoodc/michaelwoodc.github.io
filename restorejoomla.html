Grab the IP address of your windows machine
Place it in your browser on Ubuntu and download the joomla backup from your www directory. If you experience issues, I suggest you ping windows from Ubuntu to see if WAMP is reachable.
Navigate to PHP my admin and create a database named joomla
Navigate to downloads and extract the backup zip folder
Rename it simply "joomla"
Copy it
This portion MUST be done in a nautilus window with sudo priviliges. If you do not have one open, run:
sudo nautilus
Now, navigate to kickstart.php from your XAMPP front end
Hit start
You will be greeted with an error! Here is where having a nautilus window with sudo priviliges comes in handy
Find the folder "joomla" in your htdocs folder
Right click > Properties
Change your permissions as shown:
Refresh the page and retry
Now it begins with no issues! We merely needed permissions to run the kickstart.php and relevant files!
Run the installer
All default settings are ok
