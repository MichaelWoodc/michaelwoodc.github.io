Welcome!

In this hands-on project, we'll dive into the dynamic landscape of information technology, where advancements and changes can occur very quickly. In our project, we will utilize VMware Workstation, a versatile and robust virtualization platform. Our project involves installing and configuring three different operating systems: Windows 10, Windows Server 2019, and Ubuntu 22.04, then configuring and managing a virtual network between all three systems, just as you would with physical systems.

While the initial set of instructions has been created to ensure a smooth deployment of web servers, the installation and configuration of specific roles on the server, and applications on the desktop operating systems, keep in mind that the field of IT is ever-evolving. As time progresses, troubleshooting may become necessary, as processes may evolve.

During this project, we will:

  1. Install VMware and relevant operating systems onto a portable drive
  2. Simulate real network configurations and settings between our virtual machines on our virtual network
  3. Practice incremental changes in virtual machines & create periodic backups
  4. Develop a webserver with Wamp on a windows PC using the loopback address (127.0.0.1/localhost)
  5. Install and configure Joomla as a CMS (Content management system) on our Wamp server
  6. Create a hot backup of our Joomla installation
  7. Restore our backup created on Windows onto Ubuntu with XAMPP as the software stack
  8. Setup a Windows Server
  9. Configure DHCP & DNS
  10. Create and add computers to an Active Directory
