Installing Ubuntu follows the same basic steps as before, but ensure you enter the relevant credentials, you can install without installing updates at the same time, and perform a minimal installation to save time. You will likely need to input your username and password again during the setup process. Further, you can choose a minimal installation, and to install without updates, to speed up the process.
